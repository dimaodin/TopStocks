> Web Application that shows the current stock prices for Facebook, Apple, Amazon, Netflix and Google. It updates every day using the alphavantage.co API, developed in React.js.

## Installation
At the root of the project run
```
$> npm install
```

## Commands
To Run the project
```
$> npm start
```
